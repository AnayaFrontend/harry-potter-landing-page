# Universo Mágico - Harry Potter Characters

> Uma jornada interativa para explorar os personagens mais icônicos do mundo bruxo.

---

## 📖 Sobre o Projeto

Este projeto é uma **Landing Page imersiva** desenvolvida para listar e filtrar personagens da saga Harry Potter. O objetivo foi criar uma interface mágica, responsiva e performática, consumindo dados de uma API pública.

### ✨ Funcionalidades Principais

- **Listagem de Personagens:** Visualização em cards com fotos e informações.
- **Filtro por Casas:** Navegue entre Grifinória, Sonserina, Corvinal e Lufa-Lufa.
- **Busca em Tempo Real:** Encontre personagens pelo nome instantaneamente.
- **Design Responsivo:** Layout adaptado para Mobile, Tablet e Desktop.
- **Micro-interações:** Efeitos de hover, animações de entrada e transições suaves.

## 🚀 Como Executar o Projeto

Siga os passos abaixo para rodar o projeto em sua máquina local:

### Pré-requisitos

Você precisa ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### Instalação

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    ```

2.  **Acesse a pasta do projeto:**

    ```bash
    cd harry-potter-characters
    ```

3.  **Instale as dependências:**

    ```bash
    npm install
    # ou
    yarn install
    ```

4.  **Inicie o servidor de desenvolvimento:**

    ```bash
    npm run dev
    # ou
    yarn dev
    ```

5.  O projeto estará rodando em `http://localhost:5173`.

## 🎨 Layout e Design

O design foi pensado para trazer a atmosfera de Hogwarts para o navegador:

- **Paleta de Cores:** Tons escuros profundos (`#050813`) contrastando com dourado mágico (`#e6b325`).
- **Tipografia:** Fontes serifadas clássicas (`Cinzel` e `Crimson Text`) para remeter a livros antigos e feitiços.
- **Efeitos:** Uso de `backdrop-filter` para transparências modernas e sombras douradas para destaque.
